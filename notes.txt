If using 2 displays, these are fairly standard pins and allow for an efficient layout of wires:

Display 1:

SDA -> GPIO23
SCLK -> GPIO18 yellow
DC -> GPIO2 white
RST -> GPIO4 brown
CS -> GPIO5 green

Display 2:

SDA -> GPIO23 (same as Display 1)
SCLK -> GPIO18 (same as Display 1)
DC -> GPIO27 white
RST -> GPIO26 brown
CS -> GPIO15 green
